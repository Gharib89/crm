# crm — Project Memory

Python CLI for Microsoft Dynamics 365 Customer Engagement on-premises (v9.x) over the Dataverse Web API + NTLM. Single-package layout (`crm/`), pyright strict on `crm/core/*` and `crm/utils/d365_backend.py`, basic mode elsewhere.
