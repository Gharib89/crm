# Prior-Art Scenario Harvest — June 2026

This file ingests three previously-written research artifacts into the standard scenario schema so later harvest files can be diffed and merged consistently. **onprem-automation.md** (2026-05-30, two passes, 46 verified claims) surveys the D365 CE on-prem v9.x customization surface, Web API/OData v4 automation contract, and on-prem auth models, and is the primary source for platform facts and the Configuration Migration Tool (s18). **ai-agent-tooling-onprem.md** (2026-06-03, single pass, 25 verified claims) surveys the AI-tooling ecosystem across Claude Code, GitHub Copilot, Microsoft-native AI, and OSS/community MCP servers, concluding every first-party Dataverse AI tool is cloud/Entra-only and that the viable on-prem path is a generic coding agent orchestrating an NTLM-aware CLI bridge. **crm-cli-agentic-enhancements.json** (no explicit date in document; it cites ai-agent-tooling-onprem.md, dated 2026-06-03, so it postdates the two markdown files) is a structured enhancement-ideas catalogue for the crm CLI — 36 numbered recommendations (R1–R36) and a `ground` section cataloguing current capabilities and gaps across agent ergonomics, ALM authoring completeness, and on-prem/AI-tooling constraints.
